# TradeCat eval plan — 2026-05-04

> Archetype: **pure-cli**. Layer: **molecule** — 4 datasets + sync +
> probe + TUI all wired by predefined orchestration. No LLM, so the
> ceiling is determined by the molecule rule (live e2e for `reusable`),
> not by the hybrid-cap.

## What this repo claims

Pulled from `README.md` (develop branch) + `install.sh` + `pyproject.toml`
+ `dataset_registry.json` (links in `claims/claim-map.yaml`):

- 一键 curl|sh 装上 Python 3.12 CLI/TUI，自动初始化缓存，可被环境变量
  完全覆盖
- 4 个 dataset 由 `dataset_registry.json` 管理（market_snapshot /
  anomaly_panel / market_stats / event_stream）
- 桌面端 + 远端 `request.py` 共享同一个 dataset registry —— 装与不装
  数据契约一致
- TUI 在不支持 curses 的终端自动降级为 plain 文本，不抛 traceback
- 启动前自动更新有节流，可关闭，失败默认不阻断
